수업 준비와 교실 활동을 같은 자리에서

ClassPalette는 교사가 매일 반복하는 작은 작업들을 탭 기반 워크스페이스로 묶어 빠르게 전환할 수 있게 합니다.

학생 명단

반 학생 정보를 정리하고 다른 활동의 기본 데이터로 활용합니다.

기초시간표

요일별 수업과 시간 구분을 저장하고 PDF로 내보낼 수 있습니다.

오늘의 시간표

당일 수업 흐름을 확인하고 필요한 과목을 바로 조정합니다.

캘린더와 급식

학교 일정과 오늘의 급식 정보를 수업 화면 가까이에 둡니다.

자리 바꾸기

학생 배치를 시각적으로 다루며 교실 상황에 맞게 조정합니다.

뽑기 도구

랜덤 번호, 발표자, 칭찬 뽑기처럼 수업 참여를 돕는 도구를 제공합니다.

쉬는시간 타이머

수업 전환과 활동 시간을 데스크톱 화면에 크게 띄울 수 있습니다.

체육대회 점수판

행사와 활동에서 점수를 빠르게 조정하고 화면에 보여줍니다.

교실의 하루에 맞춘 작업 흐름

준비, 진행, 전환을 반복하는 실제 수업 리듬에 맞춰 구성했습니다.

1

수업 전 준비

학생 명단과 기초시간표를 정리하고 오늘 사용할 화면을 열어둡니다.

2

수업 중 전환

오늘의 시간표, 타이머, 뽑기 도구를 탭으로 빠르게 오갑니다.

3

활동 운영

자리 배치, 발표자 뽑기, 점수판으로 교실 활동을 이어갑니다.

초등교사를 위한 조용한 데스크톱 도구

ClassPalette는 Windows 교실 PC에서 쓰는 것을 기준으로 만든 Flutter 앱입니다. 복잡한 설명보다 바로 눌러 쓰는 기능, 반복 사용에 부담 없는 화면, 수업 흐름을 방해하지 않는 작업 공간을 목표로 합니다.
